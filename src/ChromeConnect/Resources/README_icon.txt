# Application Icon

This directory should contain the application icon file:
- File name: icon.ico
- Format: Windows Icon (.ico)
- Recommended sizes: 16x16, 32x32, 48x48, 64x64, 128x128, 256x256 pixels
- Color depth: 32-bit (with alpha channel)

## Creating an Icon

You can create an icon using:
1. Online icon generators (e.g., favicon.io, icoconvert.com)
2. Image editing software (GIMP, Photoshop, etc.)
3. Specialized icon editors (IcoFX, Greenfish Icon Editor Pro)

## Note

If no icon.ico file is present, the application will use the default .NET executable icon.
The icon reference in the .csproj file can be removed if not using a custom icon.
